Desenvolvimento do Movielab

O MovieLab foi desenvolvido como um projeto de Desenvolvimento Web. A ideia surgiu da paixão por filmes e séries clássicas e também da vontade de criar um site inspirado em plataformas de avaliação de filmes, onde os usuários podem conhecer novas obras e descobrir informações sobre seus títulos favoritos.

Objetivo do Projeto

O objetivo do MovieLab é reunir em um único lugar informações sobre filmes e séries marcantes, oferecendo uma navegação simples, organizada e agradável para os usuários.

Tecnologias Utilizadas

  • HTML5 para estruturar as páginas.
  • CSS3 para estilização e responsividade.
  • JavaScript para pesquisa e interatividade.
  • Font Awesome para os ícones do site.

Funcionalidades

  • Catálogo de filmes clássicos.
  • Catálogo de séries clássicas.
  • Barra de pesquisa.
  • Página de contato.
  • Design responsivo para diferentes telas.
